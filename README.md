# Kursplaneraren

Detta projekt är utvecklat med Angular och TypeScript.

Applikationen visar Mittuniversitetets kurser från en JSON-fil och gör det möjligt att söka, filtrera och sortera kurser. Användaren kan även skapa ett eget ramschema och spara det i webbläsarens localStorage.

## Funktioner

* Visa kurser från JSON-fil
* Söka efter kurser
* Filtrera kurser efter ämne
* Sortera kurser
* Visa kursplaner
* Lägga till kurser i ramschema
* Ta bort kurser från ramschema
* Beräkna totalt antal högskolepoäng
* Spara ramschema med localStorage
* Responsiv design för dator, surfplatta och mobil
